# MAHAMORDO Design Context

## Design Context

### Users
- **Primary**: Young Thai adults (18-50), mix of tech-savvy and traditional
- **Context**: Seeking fortune telling / divination — some for fun, some for real guidance
- **Job to be done**: Get a personalized, meaningful fortune reading from one of 3 AI oracle characters in a visual novel-style experience

### Brand Personality
**Mystical, Luxurious, Wise**

### Aesthetic Direction
- **Theme**: Dark mode (#0a0a0a) with cream (#FDFBF7) and gold (#D4A017, #B8860B) accents
- **Typography**: Cinzel (headings/labels) + Prompt (body/Thai) + Bebas Neue (display)
- **Anti-references**: Generic SaaS, kawaii aesthetics, cheap horoscope apps

### Design Principles
1. Atmosphere over interface
2. Earned elegance — gold used sparingly
3. Characters are the product
4. Thai identity, not decoration
5. Progressive revelation
